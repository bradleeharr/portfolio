Digital Design Lab: Dance Dance Revolution Game Project

For this project, an LPC1769 microcontroller was programmed to run a Dance Dance Revolution Game, providing visuals, music, and taking input from a PlayStation 2 Dance Pad.

Game Programming

The game was developed in C, with the LPC1769 microcontroller acting as the heart of the game system. It handles the processing and timing of the game, the display of visuals, and the playing of music.

Hardware Integration

The PlayStation 2 Dance Pad was integrated as the primary input device for the game. A driver was developed to interpret the pad's signals and convert them into game commands.

Game Experience

With a fully integrated hardware and software system, players could fully immerse themselves in the Dance Dance Revolution game, marking a successful execution of the project.
