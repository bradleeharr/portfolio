Hamming Encoder IC Design Project

The project focuses on the design, simulation, layout, and verification of a Hamming (7,4) encoder and decoder circuit. Hamming (7,4) code is widely used in digital communication systems for error detection and correction.

Design

The design phase involved creating the logic and architecture for both the Hamming (7,4) encoder and decoder. The encoder receives 4 bits of data and generates 3 parity bits, while the decoder uses these parity bits to detect and correct errors in the transmitted data.

Simulation

Simulation tests were run to confirm the functionality of the design. This phase involved generating a test bench to simulate all possible input combinations and validate the encoder and decoder's ability to accurately handle data and errors.

Layout

In the layout phase, the digital schematic of the Hamming (7,4) encoder and decoder was translated into an actual IC design. This involved creating a physical layout that accurately represents the design schematic while optimizing for space and performance.

Verification

The final phase of the project was verification, where the completed layout was checked against the original design and simulations. This was done through a process called Layout vs Schematic (LVS) to ensure that the layout accurately represented the schematic design.
